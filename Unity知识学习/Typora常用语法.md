# Typora 常用语法示例

## 标题

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 文本样式

```
**粗体文本**：ctrl+B
*斜体文本*：ctrl+I
~~删除线~~：shift+Alt+5
--下划线--：ctrl+u
```

## 列表

### 无序列表

```
- 项目1
- 项目2
  - 子项目1
  - 子项目2
```

### 有序列表

```
1. 项目1
2. 项目2
   1. 子项目1
   2. 子项目2
```

## 引用

```
> 这是引用的文本。
```

## 链接

```
[链接文本](http://www.example.com)
```

## 图片

```

```
